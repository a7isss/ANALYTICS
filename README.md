# Hail Data Observatory - Interactive Presentation

A modern, React-based alternative to PowerPoint, analyzing HRDA's real datasets dynamically.

## Features
- **Arabic First (RTL)** interface.
- **Dynamic Charts**: Interactive visualizations of Tourism, Infrastructure, and Real Estate data.
- **Storytelling Mode**: Slide transitions powered by Framer Motion.
- **Real Data**: Hardcoded insights derived from H1 2025 Source Files.

## How to Run
1. Open this folder in the terminal:
   ```bash
   cd "f:\0- Future\11- THE DATA ANALYST TRANSFER\hail-data-presentation"
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the presentation:
   ```bash
   npm run dev
   ```
4. Open the browser link shown (usually `http://localhost:5173`).

## Presentation Use Case
Use this in your interview. Instead of opening a PDF, open this website. 
Click through the "Slides" using the arrow buttons. Hover over the charts to show specific numbers (e.g., Al-Khuzama Price/m2).
